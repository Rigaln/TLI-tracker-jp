# ANTIGRAVITY.md - YiHuo ETor 日本語化プロジェクト憲法

## 1. プロジェクト概要
Torchlight: Infinite（トーチライト：インフィニティ）の多機能トラッカーツール「易火-ETor (YiHuo ETor)」を日本語化し、日本のプレイヤーが公式ゲーム用語で快適に利用できるようにするプロジェクト。

## 2. 開発戦略：MVP（最小構成）の範囲
- **UIローカライズ**: メイン画面、設定画面、通知、ダイアログの完全日本語化。
- **データローカライズ**: ゲーム内アイテム名、マップ名、カテゴリ名の日本語化（公式日本語訳に準拠）。
- **非破壊編集**: 元の `app.asar` を直接破壊せず、作業用ディレクトリで編集・検証を行う。

## 3. 技術スタック
- **Core**: Electron (Node.js)
- **Archive**: ASAR (npx asar)
- **Format**: JSON / JavaScript (翻訳定義ファイル)

## 4. 現在のステップ
- [ ] STEP 1: `app.asar` の解凍と内部構造の解析
- [ ] STEP 2: 既存の英語翻訳データの抽出と日本語翻訳ファイルの作成
- [ ] STEP 3: 公式ゲーム用語の適用（データ部分の反映）
- [ ] STEP 4: 再パッケージングと動作確認（味見）

## 5. 注意事項
- 翻訳にはゲーム内の公式用語を優先して使用する。
- 破壊的な変更（元のファイルの直接上書きなど）の前に必ず確認を行う。
